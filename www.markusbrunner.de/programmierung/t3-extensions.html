TYPO3-Extensions (im TER veröffentlicht)

mbkdimagelightbox

Eine einfach einsetzbare und multifunktionale Bildergalerie, die auch im Internet-Agentur-Alltag zum Standardwerkzeug dazugehört. Von einer einfachen Lightbox-Lösung über die allseits bekannte smoothgallery bis hin zu der Inline-Bildergalerie Pop-Eye ist hier alles integriert, was das Herz begehrt. Die Extension ist in beinahe allen Belangen, die man sich nur wünschen kann, über TypoScript konfigurierbar. Aber schaut selbst... [die Extension ist auf meiner Website in mehreren Modi im Einsatz, siehe Fotografie; im TER] - Vorstellung der Extension

mbkdvideo

Videos sind total im kommen und genauso HTML5. Diese Extension ermöglicht die Einbettung von HTML5-Videos oder -Audios. Sollte der Browser kein video- oder audio-Tag beherrschen, oder die Mime-Types der Dateien nicht unterstützen greift ein Flash-Fallback. Alternativ kann das Video aber auch direkt als Flash eingebunden werden. Als Standard-Player ist der flowplayer integriert. Mittels Template-Überschreibung (TypoScript) ist aber jeder Flash-Player integrierbar, ohne dass die Extension ihre Update-Fähigkeit verliert. [die Extension ist auf meiner Website innerhalb der Projekte-Extension und der Songlist-Extension sowie direkt im Einsatz; im TER] - Vorstellung der Extension

mbkdprevnextpaging

Hiermit kann man eine Vor- Zurück-Navigatin für Seiten oder Extensions integrieren. Die Navigation kann wiederum entweder einen statischen oder dynamischen Text besitzen (Seitenname, Datensatzname). [die Extension ist auf meiner Website im Einsatz, siehe Seitennavigation oben; im TER]

mbpageandextensionsitemap

Mit dieser Sitemap-Extension ist es möglich auch Datensätze aus Extensions in der Sitemap mit abzufangen. Beispielweise die tt_news-Datensätze können hiermit in der Sitemap erfasst werden. Eine Limitierung auf eine festlegbare Anzahl limitiert die Anzahl der Datensätze. Eine Ajax-Navigation kann dafür sorgen, dass alle weiteren Datensätze per Paging aber auch abgerufen werden können. [die Extension ist auf meiner Website im Einsatz, siehe Sitemap; im TER]

mbviralmail

Hiermit kann man die Funktionalität "Seite empfehlen" in die Seite integrieren. Dieses Plugin öffnet einen Overlay-Layer, in dem ein Formular zum Versand der Empfehlung eingebaut ist. [die Extension ist auf meiner Website im Einsatz, siehe Linktip im Footer; im TER]

mbie6boykott

Mein Beitrag zur Vernichtung des veralteten IE6-Browsers. Die meisten Benutzers wissen ja gar nicht, dass sie einen veralteten Browser einsetzen. Warum sollte man sie deshalb nicht darauf hinweisen. Die Extension bietet verschiedene Modi, um dies zu tun. Von dezenten Hinweisen bis zur Komplettabschaltung der Website-Ausgabe. Die Ausgabevariante und deren Infotexte sind per TypoScript leicht anpassbar. [die Extension ist auf meiner Website im Einsatz; natürlich nur, wenn man die Website mit dem IE6 anschaut; im TER]

google_jsapi

Mit dieser Extension kann man spielend einfach die Google JS-API einbinden. In einer Extension kann man dann die benötigte Library (jQuery, mootools) über statische Funktionen der Extensionklasse einbinden lassen. Dies sorgt dafür, dass die Library auch garantiert nur einmal im gesamten System eingebunden wird, auch wenn die Einbindung durch die Extension erfolgt! [im TER]

mbkdcssstylesheetswitcher

Diese Extension kann verschiedene CSS-Design-Varianten als Datensätze verwalten und ermöglicht dem Benutzer im Frontend das von ihm präferierte Design auszuwählen. [die Extension ist auf der Website von Katja Deutschmann im Einsatz (www.katjadeutschmann.de); im TER]

nfchighlightnav

Hiermit kann eine Erlebnisnavigation ausgegeben werden. Die Tabelle "pages" von TYPO3 wird hierfür je um ein Bild (normal und hover) erweitert. Statt Texten werden die Bilder als Navigation ausgegeben und dies aber durcheinander. Fährt man über ein Bild, werden alle Bilder der gleichen Kategorie gehighlighted. Wer mehr hierzu wissen will, sollte sich einfach mal die Dokumentation dazu durchlesen. Diese veranschaulicht die Thematik leicht verständlich. [im TER]

TYPO3-Extensions (nicht im TER)

mbwikifileknowledgebase

Diese Extension stellt die Umsetzung meines Medienlexikons "MediaLex" dar. Textdateien in Wikisyntax werden als Datensätze aufgenommen. Diese stellen wiederum einen Beitrag dar, der im Frontend als HTML geparst ausgegeben wird. Desweiteren wird ein Link auf die originale Textdatei ausgeliefert. Das geparste HTML wird in der Datenbank gecacht und nur bei Überschreibung der Textdatei (neuer Zeitstempel) neu generiert. [die Extension ist auf meiner Website im Einsatz, siehe MediaLex]

mbprojects

Diese Extension stellt die Umsetzung meiner Projekteausgabe dar. Projektdatensätze, die Kategorien zugeordnet werden, können als Liste und Detailansicht ausgegeben werden. Auf der Detailseite wird aus der Extension heraus die mbkdimagelightbox eingebunden, um die Projektbilder darzustellen. [die Extension ist auf meiner Website im Einsatz, siehe Projekte]

mbsonglist

Diese Extension stellt die Umsetzung meiner Audiografie-Interpreten-Ausgabe dar. Einem Interpreten können Alben und diesen wiederum Titel zugeordnet werden. Diese besitzen Audio-Dateien, Lyrics, Videos, Tabs. Aber schaut doch am besten selbst... [die Extension ist auf meiner Website im Einsatz, siehe Audiografie]
